# Category Icon Asset Prompts

## Style Primer

Prefix every prompt with:

> Flat vector icon on a transparent background, warm earthy color palette (browns, ambers, creams), rounded soft edges, minimal detail, 256x256px, suitable for a book collection app. No text.

## Specs

- **Size:** 256x256px (displayed at 48x48 in-app)
- **Format:** PNG with transparent background
- **Naming:** `icon_<category>.png` (e.g., `icon_art.png`)
